# Function

## 文件说明

|文件|说明|
|---|---|
|index.js| 主函数文件，包含handler(默认入口函数), initializer(默认初始化函数)|
|config.js|默认配置文件|
|template.yml|函数计算Fun工具配置文件|
|test.spec.js|测试文件|


## 开发

> `npm start` 需要fun工具local的能力支持。 

```
npm i
npm start
```

  

## 测试

```
npm run test
```

## 版本管理

```
npm run release
```
